# NodeJS Prisma REST API

## Description

A Simple REST API Backend using Prisma ORM, PostgreSQL & ExpressJS.

## Installation

```sh
$ npm install
```

## Running the app

```sh
$ node server
```

## Setup

Create .env file in the root directory and add the following configuations:

```
DATABASE_URL - Set database URL String
```
